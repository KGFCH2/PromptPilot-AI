# Security Policy

## Supported Versions

Only the latest release version on the `main` branch is actively supported with security updates.

| Version | Supported |
| ------- | --------- |
| 1.0.x   | Yes       |
| < 1.0   | No        |

## Reporting a Vulnerability

If you discover a potential security vulnerability in PromptPilot AI, please **do not** open a public issue. Instead, report it responsibly by contacting the maintainers directly or emailing security@promptpilot-ai.example.org.

Once a vulnerability report is received, the maintainers will:
1. Acknowledge receipt of the report within 48 hours.
2. Work on a resolution in an isolated security patch branch.
3. Release a patch version and publish a security advisory.

Thank you for helping keep this project secure!
